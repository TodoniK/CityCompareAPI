Webservices de notre API


Cette API permet de comparer différentes villes de France (à partir du code postal que vous fournissez), selon plusieurs critères :

  • La température de la ville
  • L'indice de qualité de l'air
  • Le prix moyen des différents carburant disponibles
Notre API renvoi donc les valeurs des différents critères des villes et calcule un score en fonction de celles-ci.
Le webservice quant à lui, permet soit d'afficher le JSON brut de notre API; ou bien de traiter le JSON afin d'afficher un graphique, plus compréhensible.
